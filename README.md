# Mini-Games-Collection

A collection of small games I built while learning **C++** and the
**SplashKit** graphics library during my first year of Computer Science at
Monash University.

Each game lives in its own folder with its own README explaining what it does
and how to run it. 

## Games

| Game | Description |
|------|-------------|
| [Click Game](click-game) | A reflex game — click randomly placed targets to fill a progress bar and win. |
| [Reaction Game](reaction-game) | Click targets as fast as you can — times each click and rates your reflexes. |
| [Rock Dodge Game](rock-dodge-game) | An arcade dodging game — move to avoid falling rocks and survive as long as you can. |


## Built with

- **C++**
- **SplashKit** — a graphics and game library used for windows, drawing, and input

## Running the games

Each game folder has its own README with specific build and run instructions.
In general you'll need [SplashKit](https://splashkit.io) installed, then compile
from inside that game's folder.
